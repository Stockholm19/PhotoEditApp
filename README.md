# üì± PhotoEditApp

**PhotoEditApp** is a simple iOS application for image editing with basic authentication.  
It was developed as a test project using **SwiftUI**, **Firebase**, and the **MVVM** architecture.

---

## ‚ú® Features

- ‚úÖ Email/password registration and login  
- ‚úÖ Basic form validation (email and password)  
- ‚úÖ Password reset (placeholder)  
- ‚úÖ Select an image from the photo library  
- ‚úÖ Save the selected image to `UserDefaults`  
- ‚úÖ Image persists between sessions  
- ‚úÖ Replace or delete the image  
- ‚úÖ Scale, rotate, and move the image using gestures  
- ‚úÖ Add text overlay with drag support  
- ‚úÖ Drawing canvas (PencilKit, currently not bound to image)  
- ‚úÖ Save the final edited image to the photo library
